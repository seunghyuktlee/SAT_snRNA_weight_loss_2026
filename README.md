# SAT_snRNA_weight_loss_2026
This repository contains the code used for analyses presented in the publication "Longitudinal adipose tissue single cell transcriptomics reveals genes and variants regulating weight loss after bariatric surgery".

Longitudinal adipose tissue single cell transcriptomics reveals genes and variants regulating weight loss after bariatric surgery. medRxiv [Preprint]. 2025 Jul 14:2025.07.11.25331390.
